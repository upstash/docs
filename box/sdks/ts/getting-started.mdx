The Upstash Box TypeScript SDK provides a fully typed client for interacting with Upstash Box.

## Installation

<CodeGroup>
```bash npm
npm install @upstash/box
```

```bash yarn
yarn add @upstash/box
```

```bash pnpm
pnpm add @upstash/box
```

```bash bun
bun install @upstash/box
```
</CodeGroup>

## Quick Start

```typescript
import { Box } from "@upstash/box"

const box = new Box({
  url: process.env.UPSTASH_BOX_REST_URL,
  token: process.env.UPSTASH_BOX_REST_TOKEN,
})

await box.upload("hello.txt", "Hello, World!")

const content = await box.get("hello.txt")
```

## Configuration

| Option | Type | Description |
| --- | --- | --- |
| `url` | `string` | Your Box REST URL from the Upstash Console |
| `token` | `string` | Your Box REST token from the Upstash Console |
