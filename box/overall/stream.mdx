---
title: "Stream"
---

`box.agent.stream()` returns an async generator that yields typed chunks in real time as the agent works. You get text deltas, reasoning traces, tool calls, and usage stats as they happen instead of waiting for the full response.

---

### Stream agent output

Call `box.agent.stream()` with a prompt. Iterate over the returned async generator to receive chunks as the agent produces them.

```typescript
for await (const chunk of box.agent.stream({ prompt: "Add input validation" })) {
  if (chunk.type === "text-delta") {
    process.stdout.write(chunk.text)
  }
}
```

---

### Chunk types

Every chunk has a `type` field you can switch on. The full set of chunk types:

| Type         | Fields                         | Description                                                 |
| ------------ | ------------------------------ | ----------------------------------------------------------- |
| `start`      | `runId`                        | Emitted once when the run begins                            |
| `text-delta` | `text`                         | A fragment of the agent's text output                       |
| `reasoning`  | `text`                         | A fragment of the agent's internal reasoning                |
| `tool-call`  | `toolName`, `input`            | The agent invoked a tool (Read, Write, Bash, etc.)          |
| `finish`     | `output`, `usage`, `sessionId` | The run completed and includes full output and token counts |
| `stats`      | `cpuNs`, `memoryPeakBytes`     | Resource usage for the run                                  |

```typescript
for await (const chunk of box.agent.stream({ prompt: "Refactor auth module" })) {
  switch (chunk.type) {
    case "start":
      console.log(`Run started: ${chunk.runId}`)
      break
    case "text-delta":
      process.stdout.write(chunk.text)
      break
    case "reasoning":
      console.log(`[thinking] ${chunk.text}`)
      break
    case "tool-call":
      console.log(`Tool: ${chunk.toolName}`, chunk.input)
      break
    case "finish":
      console.log(
        `\nTokens: ${chunk.usage.inputTokens} in / ${chunk.usage.outputTokens} out`,
      )
      break
    case "stats":
      console.log(`CPU: ${chunk.cpuNs}ns, Memory peak: ${chunk.memoryPeakBytes} bytes`)
      break
  }
}
```

---

### Callbacks

If you prefer callbacks over an async generator, `box.agent.stream()` accepts `onChunk` and `onToolUse` handlers that fire alongside the generator.

```typescript
for await (const chunk of box.agent.stream({
  prompt: "Write unit tests",
  onChunk: (chunk) => console.log(chunk.type),
  onToolUse: (tool) => console.log(`Agent used ${tool.name}`),
})) {
  // chunks are also yielded here
}
```

`box.agent.run()` also supports a simpler `onStream` callback that receives text-only strings as the agent writes:

```typescript
const run = await box.agent.run({
  prompt: "Fix the failing tests",
  onStream: (text) => process.stdout.write(text),
})

console.log(run.output)
```

---

### Timeout

Pass `timeout` in milliseconds to abort a stream that runs too long. The SDK throws a `BoxError` when the timeout is exceeded.

```typescript
for await (const chunk of box.agent.stream({
  prompt: "Migrate the database schema",
  timeout: 120_000,
})) {
  if (chunk.type === "text-delta") process.stdout.write(chunk.text)
}
```

## Patterns

### Live CLI output

Stream agent output directly to the terminal for an interactive developer experience.

```typescript
import { Box, ClaudeCode } from "@upstash/box"

const box = await Box.create({
  runtime: "node",
  agent: { model: ClaudeCode.Sonnet_4_5, apiKey: process.env.ANTHROPIC_API_KEY },
})

for await (const chunk of box.agent.stream({ prompt: "Explain the project structure" })) {
  if (chunk.type === "text-delta") {
    process.stdout.write(chunk.text)
  }
}

await box.delete()
```

### Track tool usage

Monitor which tools the agent invokes to build audit logs or progress indicators.

```typescript
import { Box, ClaudeCode } from "@upstash/box"

const box = await Box.create({
  runtime: "node",
  agent: { model: ClaudeCode.Opus_4_6, apiKey: process.env.ANTHROPIC_API_KEY },
  git: { token: process.env.GITHUB_TOKEN },
})

await box.git.clone({ repo: "github.com/your-org/your-api" })

const tools: { name: string; input: Record<string, unknown> }[] = []

for await (const chunk of box.agent.stream({
  prompt: "Find and fix the memory leak in src/cache.ts",
  onToolUse: (tool) => tools.push(tool),
})) {
  if (chunk.type === "text-delta") process.stdout.write(chunk.text)
}

console.log(`\nAgent used ${tools.length} tool calls:`)
tools.forEach((t) => console.log(`  ${t.name}`))
```

### Collect the full output while streaming

Accumulate text deltas into a complete string while still printing in real time.

```typescript
import { Box, ClaudeCode } from "@upstash/box"

const box = await Box.create({
  runtime: "node",
  agent: { model: ClaudeCode.Sonnet_4_5, apiKey: process.env.ANTHROPIC_API_KEY },
})

let fullOutput = ""

for await (const chunk of box.agent.stream({
  prompt: "Write a README for this project",
})) {
  if (chunk.type === "text-delta") {
    process.stdout.write(chunk.text)
    fullOutput += chunk.text
  }
  if (chunk.type === "finish") {
    console.log(`\n\nTokens used: ${chunk.usage.inputTokens + chunk.usage.outputTokens}`)
  }
}

await box.files.write("/work/README.md", fullOutput)
await box.delete()
```
