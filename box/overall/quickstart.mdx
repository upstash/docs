---
title: "Quickstart"
---

Upstash Box lets you run AI agents in secure, isolated cloud containers with Claude Code built-in. Each box is a full environment with a filesystem, git, and a runtime. Your agent can read files, write code, and execute tasks inside it.

---

### 1. Get your API key

Navigate to the [Upstash Console](https://console.upstash.com) and create a new Box API key.

<Frame>
  <img src="/img/box/create-api-key.png" />
</Frame>

### 2. Install the SDK

<CodeGroup>
```bash npm
npm install @upstash/box
```

```bash yarn
yarn add @upstash/box
```

```bash pnpm
pnpm add @upstash/box
```

```bash bun
bun install @upstash/box
```

</CodeGroup>

### 3. Set Environment Variables	

Add your Upstash Box API key:

```bash title=".env"
UPSTASH_BOX_API_KEY=abx_xxxxxxxxxxxxxxxxxxxxxxxx
```

### 4. Create a Box

```typescript title="lib/box.ts"
import { Box } from "@upstash/box"

const box = await Box.create({
  runtime: "node",
})
```

### 5. Configure the Agent

All boxes come with built-in Claude Code. We expanding to other code harnesses like OpenCode soon, too.

To configure Claude Code for your box, pass the model you want to use and your Anthropic API key:

```typescript {5-8}
import { Box, ClaudeCode } from "@upstash/box"

const box = await Box.create({
  runtime: "node",
  agent: {
    model: ClaudeCode.Sonnet_4_5,
    apiKey: process.env.ANTHROPIC_API_KEY,
  },
})
```

### 6. Run Your First Task

Your box is ready to use! It's a secure cloud environment to run agents, execute commands, or manage files. For example:

```typescript
import { Box, ClaudeCode } from "@upstash/box"

const box = await Box.create({
  runtime: "node",
  agent: {
    model: ClaudeCode.Sonnet_4_5,
    apiKey: process.env.ANTHROPIC_API_KEY,
  },
})

// Execute OS-level commands
await box.exec("node --version")

// Run Claude Code
await box.agent.run({
  prompt: "create an index.txt saying 'hello world'",
})
```
