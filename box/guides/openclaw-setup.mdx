This guide walks you through setting up [OpenClaw](https://docs.openclaw.ai) inside an Upstash Box.

---

## 1. Create a box

Create a box with default settings. See the [quickstart](/box/overall/quickstart) if you haven't created one before.

## 2. Connect via SSH

Once the box is running, connect to it via SSH from your terminal. Use your [Box API key](/box/overall/quickstart#1-get-your-api-key) as the password when prompted.

<img src="/img/openclaw-setup/ssh-connection.png" />

---

## 3. Install OpenClaw

Install the OpenClaw CLI globally inside the box.

```bash
sudo npm install -g openclaw@2026.3.24
```

<Note>
  `2026.3.24` is the latest stable version. The newest release has known issues — we recommend staying on this version until they are resolved.
</Note>

---

## 4. Run onboarding

Start the interactive onboarding wizard. The CLI will guide you through provider selection, authentication, and gateway configuration.

```bash
openclaw onboard --install-daemon
```

Follow the prompts to complete your setup.

---

## 5. Start the gateway

Once onboarding is complete, start the OpenClaw gateway.

```bash
openclaw gateway run
```
