---
title: "Restart Kafka Connector"
description: "This endpoint restarts an existing connector."
api: "POST https://api.upstash.com/v2/kafka/connector/{id}/restart"
authMethod: "basic"
---

## URL Parameters

<ParamField path="id" type="string" required>
  The ID of the Kafka Connector to be restarted
</ParamField>

<RequestExample>

```shell curl
curl -X POST \
  https://api.upstash.com/v2/kafka/connector/:id/restart \
  -u 'EMAIL:API_KEY'
```

```python Python
import requests

response = requests.post('https://api.upstash.com/v2/kafka/connector/:id/restart', auth=('EMAIL', 'API_KEY'))
response.content
```

```go Go
client := &http.Client{}
req, err := http.NewRequest("POST", "https://api.upstash.com/v2/kafka/connector/:id/restart", nil)
if err != nil {
    log.Fatal(err)
}
req.SetBasicAuth("email", "api_key")
resp, err := client.Do(req)
if err != nil {
    log.Fatal(err)
}
bodyText, err := ioutil.ReadAll(resp.Body)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("%s\n", bodyText);
```

</RequestExample>

<ResponseExample>

```json 200 OK
"OK"
```
