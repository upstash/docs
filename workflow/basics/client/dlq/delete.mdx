---
title: "client.dlq.delete"
---

The `dlq.delete` method removes one or more entries from the **Dead Letter Queue (DLQ)**.

## Arguments

The method accepts different argument types:

<ParamField body="request" type="string">
    A single DLQ entry ID to delete.
</ParamField>

<ParamField body="request" type="string[]">
    An array of DLQ entry IDs to delete.
</ParamField>

<ParamField body="request" type="object (dlqIds)">
    An object containing `dlqIds` to delete.

    <Expandable defaultOpen>
        <ParamField body="dlqIds" type="string | string[]">
            One or more DLQ entry IDs.
        </ParamField>
    </Expandable>
</ParamField>

<ParamField body="request" type="object (filters)">
    A filter object to delete matching DLQ entries.

    <Expandable defaultOpen>
        <ParamField body="label" type="string">
            Delete entries with this label.
        </ParamField>

        <ParamField body="fromDate" type="string">
            Delete entries created after this date (Unix ms as string).
        </ParamField>

        <ParamField body="toDate" type="string">
            Delete entries created before this date (Unix ms as string).
        </ParamField>

        <ParamField body="workflowUrl" type="string">
            Delete entries matching this workflow URL.
        </ParamField>

        <ParamField body="workflowRunIds" type="string[]">
            Delete entries matching these workflow run IDs.
        </ParamField>
    </Expandable>
</ParamField>

## Response

<ResponseField name="deleted" type="number">
    The number of DLQ entries that were deleted.
</ResponseField>

## Usage

<CodeGroup>
```ts Single ID
await client.dlq.delete("dlq-12345");
```
```ts Multiple IDs
await client.dlq.delete(["dlq-12345", "dlq-67890"]);
```
```ts Object with dlqIds
await client.dlq.delete({
  dlqIds: ["dlq-12345", "dlq-67890"],
});
```
```ts With Filters
await client.dlq.delete({
  label: "my-workflow-label",
  fromDate: "1640995200000",
  toDate: "1672531200000",
});
```
</CodeGroup>
