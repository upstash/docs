---
title: "client.dlq.delete"
---

The `dlq.delete` method deletes one or more workflow runs from the **Dead Letter Queue (DLQ)**.

## Arguments

You can delete DLQ entries by ID, by an array of IDs, or by filters.

### By DLQ ID

<ParamField body="dlqId" type="string | string[]">
    A single DLQ entry ID, or an array of IDs to delete.
</ParamField>

### By filters

<ParamField body="workflowRunId" type="string" optional>
    Filter by workflow run ID.
</ParamField>

<ParamField body="workflowUrl" type="string" optional>
    Filter by exact workflow URL.
</ParamField>

<ParamField body="fromDate" type="Date | number | string" optional>
    Delete workflows created after this date. Accepts `Date` objects, Unix timestamps (number), or numeric strings.
</ParamField>

<ParamField body="toDate" type="Date | number | string" optional>
    Delete workflows created before this date. Accepts `Date` objects, Unix timestamps (number), or numeric strings.
</ParamField>

<ParamField body="label" type="string" optional>
    Delete workflows with this label.
</ParamField>

<ParamField body="all" type="boolean" optional>
    Delete all DLQ entries.
</ParamField>

## Response

<ResponseField name="deleted" type="number">
    The number of DLQ entries that were deleted.
</ResponseField>

## Usage

<CodeGroup>
```ts Single
await client.dlq.delete("dlq-123");
```
```ts Multiple
await client.dlq.delete(["dlq-123", "dlq-456"]);
```
```ts Object with IDs
await client.dlq.delete({ dlqIds: ["dlq-123", "dlq-456"] });
```
```ts By filters
// delete by label
await client.dlq.delete({ label: "my-label" });

// delete by URL
await client.dlq.delete({ workflowUrl: "https://example.com/workflow" });

// delete by date range
await client.dlq.delete({
  fromDate: 1640995200000,
  toDate: 1672531200000,
});

// delete all
await client.dlq.delete({ all: true });
```
</CodeGroup>
