---
title: "Get Global Parallelism"
description: "Get global parallelism usage"
api: "GET https://qstash.upstash.io/v2/globalParallelism"
authMethod: "bearer"
---

## Response

<ResponseField name="parallelismMax" type="integer">
  The configured maximum global parallelism.
</ResponseField>

<ResponseField name="parallelismCount" type="integer">
  The current number of messages running globally in parallel.
</ResponseField>

<RequestExample>
```sh
curl -X GET https://qstash.upstash.io/v2/globalParallelism -H "Authorization: Bearer <token>"
```
</RequestExample>

<ResponseExample>
```json
{
  "parallelismMax": 500,
  "parallelismCount": 42
}
```
</ResponseExample>
