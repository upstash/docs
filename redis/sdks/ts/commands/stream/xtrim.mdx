---
title: XTRIM
description: Trims the stream by removing entries to keep it at a reasonable size.
---

## Arguments

<ParamField body="key" type="string" required>
  The key of the stream.
</ParamField>

<ParamField body="strategy" type="object" required>
  The trimming strategy. Can be one of:
  
  <ParamField body="MAXLEN">
    <ParamField body="type" type="'MAXLEN'" required>
      Trim based on the maximum number of entries.
    </ParamField>
    <ParamField body="threshold" type="number" required>
      The maximum number of entries to keep.
    </ParamField>
    <ParamField body="comparison" type="'~' | '='" required>
      Use '~' for approximate trimming (more efficient) or '=' for exact trimming.
    </ParamField>
    <ParamField body="limit" type="number">
      Limit how many entries will be trimmed at most (only valid with '~').
    </ParamField>
  </ParamField>

  <ParamField body="MINID">
    <ParamField body="type" type="'MINID'" required>
      Trim based on the minimum ID.
    </ParamField>
    <ParamField body="threshold" type="string" required>
      The minimum ID to keep. Entries with IDs lower than this will be removed.
    </ParamField>
    <ParamField body="comparison" type="'~' | '='" required>
      Use '~' for approximate trimming (more efficient) or '=' for exact trimming.
    </ParamField>
    <ParamField body="limit" type="number">
      Limit how many entries will be trimmed at most (only valid with '~').
    </ParamField>
  </ParamField>
</ParamField>

## Response

<ResponseField type="number">
  The number of entries removed from the stream.
</ResponseField>

<RequestExample>
```ts Approximate trim
const result = await redis.xtrim("mystream", {
  type: "MAXLEN",
  threshold: 100,
  comparison: "~"
});
```

```ts Exact trim
const result = await redis.xtrim("mystream", {
  type: "MAXLEN", 
  threshold: 50,
  comparison: "="
});
```

```ts Trim by minimum ID
const result = await redis.xtrim("mystream", {
  type: "MINID",
  threshold: "1638360173533-0",
  comparison: "="
});
```

```ts Trim with limit
const result = await redis.xtrim("mystream", {
  type: "MAXLEN",
  threshold: 1000,
  comparison: "~",
  limit: 100
});
```
</RequestExample>
