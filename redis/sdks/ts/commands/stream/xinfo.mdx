---
title: XINFO
description: Returns information about streams, consumer groups, and consumers.
---

## Arguments

<ParamField body="key" type="string" required>
  The key of the stream.
</ParamField>

<ParamField body="subcommand" type="object" required>
  The XINFO subcommand and its parameters. Can be one of:
  
  <ParamField body="STREAM">
    <ParamField body="type" type="'STREAM'" required>
      Get information about the stream.
    </ParamField>
    <ParamField body="options">
      <ParamField body="full" type="boolean">
        Return full information including entries.
      </ParamField>
      <ParamField body="count" type="number">
        Limit the number of entries returned (requires full: true).
      </ParamField>
    </ParamField>
  </ParamField>

  <ParamField body="GROUPS">
    <ParamField body="type" type="'GROUPS'" required>
      List all consumer groups for the stream.
    </ParamField>
  </ParamField>

  <ParamField body="CONSUMERS">
    <ParamField body="type" type="'CONSUMERS'" required>
      List all consumers in a consumer group.
    </ParamField>
    <ParamField body="group" type="string" required>
      The consumer group name.
    </ParamField>
  </ParamField>
</ParamField>

## Response

<ResponseField>
  The return type depends on the subcommand:
  - STREAM: Returns detailed information about the stream
  - GROUPS: Returns an array of consumer group information
  - CONSUMERS: Returns an array of consumer information
</ResponseField>

<RequestExample>
```ts Stream info
const result = await redis.xinfo("mystream", {
  type: "STREAM"
});
```

```ts Full stream info
const result = await redis.xinfo("mystream", {
  type: "STREAM",
  options: { full: true, count: 10 }
});
```

```ts List groups
const result = await redis.xinfo("mystream", {
  type: "GROUPS"
});
```

```ts List consumers
const result = await redis.xinfo("mystream", {
  type: "CONSUMERS",
  group: "mygroup"
});
```
</RequestExample>
