---
title: FUNCTION DELETE
description: Delete a library and all its functions.
---

## Arguments

<ParamField body="libraryName" type="string" required>
  The name of the library to delete.
</ParamField>

## Response

<ResponseField type="string" required>
  "OK"
</ResponseField>

<RequestExample>
```ts Example
await redis.functions.delete("mylib")
```
</RequestExample>
