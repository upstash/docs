---
title: XTRIM
description: Trims the stream by removing entries to keep it at a reasonable size.
---

## Arguments

<ParamField body="key" type="str" required>
  The key of the stream.
</ParamField>

<ParamField body="maxlen" type="int">
  The maximum number of entries to keep in the stream.
</ParamField>

<ParamField body="approximate" type="bool">
  Use approximate trimming (more efficient). When True, Redis may keep slightly more entries than specified.
</ParamField>

<ParamField body="minid" type="str">
  The minimum ID to keep. Entries with IDs lower than this will be removed.
</ParamField>

<ParamField body="limit" type="int">
  Limit how many entries will be trimmed at most.
</ParamField>

## Response

<ResponseField type="int">
  The number of entries removed from the stream.
</ResponseField>

<RequestExample>
```py Approximate trim
result = redis.xtrim("mystream", maxlen=50, approximate=True)
```

```py Exact trim
result = redis.xtrim("mystream", maxlen=20, approximate=False)
```

```py Trim by minimum ID
result = redis.xtrim("mystream", minid="1638360173533-0")
```

```py Trim with limit
result = redis.xtrim("mystream", maxlen=1000, approximate=True, limit=100)
```
</RequestExample>
