---
title: GETSET
description: Return the value of the specified key and replace it with a new value.
---


## Arguments 

<ParamField body="key" type="string" required>
  The key to get.
</ParamField>

<ParamField body="newValue"  required>
  The new value to store.
</ParamField>

## Response
<ResponseField  required>

The response is the value stored at the key or `null` if the key doesn't exist.
</ResponseField>

<RequestExample>
```py Example

oldValue = await redis.getset("key", newValue)
```
</RequestExample>
