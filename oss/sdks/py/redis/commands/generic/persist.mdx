---
title: PERSIST
description: Remove any timeout set on the key.
---


## Arguments

<ParamField body="key" type="string" required>
  The key to persist
</ParamField>

## Response

<ResponseField type="integer" required>
  `1` if the timeout was removed, `0` if `key` does not exist or does not have an associated timeout.
</ResponseField>

<RequestExample>
```py Example
redis.persist(key)
```
</RequestExample>
