---
title: HLEN
description: Returns the number of fields contained in the hash stored at key.
---

## Arguments

<ParamField body="key" type="string" required>
  The key of the hash.
</ParamField>


## Response

<ResponseField type="integer" required>
  How many fields are in the hash.
</ResponseField>

<RequestExample>
```py Example
redis.hset("key", {
  id: 1,
  username: "chronark",
  })
fields = redis.hlen("key")
print(fields); # 2
```
</RequestExample>
