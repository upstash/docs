---
title: HGETALL
description: Retrieves all fields from a hash.
---

## Arguments

<ParamField body="key" type="string" required>
  The key to get.
</ParamField>

## Response

<ResponseField type="TValue | null" required>
  An object with all fields in the hash.
</ResponseField>

<RequestExample>
```py Example
await redis.hset("key", {
  field1: "value1",
  field2: "value2",
  })
hash = await redis.hgetall("key")
print(hash); # { field1: "value1", field2: "value2" }
```
</RequestExample>
