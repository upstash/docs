---
title: "Delete Topic"
description: "This endpoint deletes an existing topic from QStash."
api: "DELETE https://qstash.upstash.io/v1/topics/{topic}"
authMethod: "bearer"
---

## Request

<ParamField path="topic" required>
  The id or name of the topic
</ParamField>

<RequestExample>

```shell curl
curl -X DELETE https://qstash.upstash.io/v1/topics/:topicId \
  -H "Authorization: Bearer <token>" \
```

</RequestExample>

<ResponseExample>

```json 200
{ "status": "ok" }
```

</ResponseExample>
