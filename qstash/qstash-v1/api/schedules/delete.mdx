---
title: "Delete Schedule"
description: "This endpoint deletes a schedule."
api: "DELETE https://qstash.upstash.io/v1/schedules/{scheduleId}"
authMethod: "bearer"
---

<Note>
  It can take up to 60s for the schedule to be deleted from the system.
</Note>

## Request

<ParamField path="scheduleID" type="string" required>
  The unique id of the schedule you want to delete
</ParamField>

<RequestExample>

```shell curl
curl -XDELETE https://qstash.upstash.io/v1/schedules/scd_1234 \
  -H "Authorization: Bearer <token>"
```

</RequestExample>

<ResponseExample>

```json 202
{ "status": "accepted" }
```

</ResponseExample>
